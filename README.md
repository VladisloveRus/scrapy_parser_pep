#  Парсер документов PEP

Проект для парсинга документов PEP

## Технологии
- Python
- Scrapy

## Использование

- Установите зависимости из директории
```sh
pip install -r requirements.txt
```
- Запустите "паучка", он всё сделает сам
```sh
scrapy crawl pep
```
- Результаты хранятся в папке results
